🎯 Aim

To design and develop a simple AI-based travel assistant chatbot using web technologies and Python that provides users with hotel and flight information.

---

📖 Description

This project is a web-based chatbot application built using Flask (Python) for the backend and HTML, CSS, and JavaScript for the frontend.

The chatbot:

- Accepts user input through a chat interface
- Processes requests using keyword-based logic
- Responds with hotel suggestions and flight details
- Supports cities like Paris, London, Tokyo, and Vietnam

🚀 How to Run

1. Install Flask:

pip install flask

2. Run the application:

python app.py

3. Open in browser:

http://127.0.0.1:5000/
